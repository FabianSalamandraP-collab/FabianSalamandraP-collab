<h2 align="center">Nuestra Misión</h2>
<p align="center">
Somos un colectivo del <strong>Sur Occidente de Bosa, Bogotá</strong>, donde estamos <strong>abriendo la tecnología y la cultura</strong> a nuestra comunidad.<br/>
Queremos hacer una <strong>revolución</strong> en cómo usamos las herramientas digitales para <strong>mostrar y revivir nuestras voces de lucha</strong>.<br/>
Creemos que la <strong>tecnología es poder</strong> y que cada línea de código, cada diseño, cada proyecto es una forma de <strong>resistencia y expresión</strong>.<br/>
Desde el barrio hasta el mundo, estamos <strong>construyendo el futuro</strong> que queremos ver.
</p>
